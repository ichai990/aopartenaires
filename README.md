# AO Partenaires

Site vitrine d'AO Partenaires — apporteur d'affaires sur les marchés publics du bâtiment (CVC, plomberie, chauffage, rénovation énergétique) en Île-de-France.

## Contenu
- `index.html` — site one-page (responsive desktop / tablette / mobile), formulaire connecté à Google Sheets.
- `charte-graphique.html` — charte graphique de la marque.
- `assets/` — logo et favicon (SVG).
- `google-apps-script.gs` — script de réception des leads dans Google Sheets.

## Déploiement (GitHub Pages)
1. Repo > **Settings** > **Pages**.
2. **Source** : `Deploy from a branch`.
3. **Branch** : `main` / dossier `/ (root)` > **Save**.
4. Le site est publié sous quelques minutes à l'adresse :
   `https://ichai990.github.io/aopartenaires/`
